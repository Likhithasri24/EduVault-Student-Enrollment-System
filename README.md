# 🎓 EduVault - Student Enrollment & Fee Management System

A menu-driven Java console application that automates student enrollment, course registration, fee payment, and enrollment management using Object-Oriented Programming (OOP) principles.

## 📖 Project Overview

EduVault is a Java-based Student Enrollment & Fee Management System developed to simplify the student admission process for training institutes such as TAP Academy.

The application enables users to register students, select their highest qualification, enroll in professional courses, generate unique enrollment IDs, process fee payments, calculate remaining balances, and display complete student enrollment information through a menu-driven console interface.

The project follows Object-Oriented Programming concepts with a modular architecture, making the code reusable, maintainable, and easy to extend with additional features.
